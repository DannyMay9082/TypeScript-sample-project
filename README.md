## Based on
Angular 4, Bootstrap 4, Angular CLI and more plugins like ng2 translate

## Usage

`npm i`

`npm start`

`npm run deploy` (fork to do this)

## Features
* TypeScript
* Angular CLI
* Responsive layout
* Bootstrap 4 CSS Framework
* Sass
* Angular 4
* jQuery
* Tests with Protractor
* Github pages deploy ready
