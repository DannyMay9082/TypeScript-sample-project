## Usage

Install: `npm i`

Run: `npm start` and go to `localhost:4200`

Test: `npm run test` and `npm run e2e`

Build: `ng build --prod`

Deploy: `npm run deploy` (fork to do this)

## Features
* Angular 4
* TypeScript
* Angular CLI
* Responsive layout
* Angular Material
* Internationalization
* Sass
* Tests with Protractor
* Github pages deploy ready
